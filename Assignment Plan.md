## ΦΑΣΗ 2: UI Design & Visuals

### Βήμα 2.1 – #9 Color Palette Selection (Υποχρεωτική)

**🛠️ Tools:**
- **ChatGPT/Claude** → Για πρόταση χρωμάτων
- **Coolors.co** ή **Realtime Colors** → Για visualization
- **WebAIM Contrast Checker** → Για έλεγχο αντίθεσης

**Prompt 1 (αντιγράψτε):**
> *"Πρότεινε μια color palette (5-6 χρώματα σε HEX) για μια mobile εφαρμογή αγροτών (Agro-Sync) που χρησιμοποιείται σε εξωτερικό χώρο κάτω από έντονο ηλιακό φως. Απαιτήσεις: Υψηλή αντίθεση (WCAG AAA), ορατότητα σε direct sunlight, φυσικά/γήινα χρώματα που ταιριάζουν στο αγροτικό περιβάλλον, ξεκάθαρη διάκριση μεταξύ χρωμάτων επιτυχίας (πράσινο), προειδοποίησης (κίτρινο/πορτοκαλί), και κινδύνου (κόκκινο). Δώσε: Primary, Secondary, Background, Surface, Success, Warning, Error, Text colors, και τον λόγο αντίθεσης (contrast ratio) κάθε συνδυασμού."*

**Τι κάνεις:**
1. Πάρε τα HEX χρώματα
2. Τσέκαρε στο **WebAIM Contrast Checker** ότι τα contrast ratios είναι τουλάχιστον **4.5:1** (AA) ή **7:1** (AAA)
3. Αν κάποιο χρώμα δεν περνάει → **ΑΥΤΗ ΕΙΝΑΙ Η ΑΠΟΡΡΙΨΗ ΣΟΥ!** Γράψε: *"Το AI πρότεινε [χρώμα X] αλλά είχε contrast ratio μόνο 3.2:1. Το αντικατέστησα με [χρώμα Y] (contrast 7.1:1) γιατί στον ήλιο θα ήταν αδύνατο να διαβαστεί."*
4. Δημιούργησε **Color Styles** στο Figma (δεξί κλικ → Create Style)

**⏱️ Χρόνος:** ~30 λεπτά
