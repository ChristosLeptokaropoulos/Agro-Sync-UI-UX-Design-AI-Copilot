# Use Cases for Agro-Sync Application

## Use Case 1: AI Scanner Διάγνωσης Ασθενειών
- **Actor**: Αγρότης
- **Preconditions**:
  - Η συσκευή διαθέτει κάμερα.
  - Ενεργή σύνδεση στο διαδίκτυο (εκτός αν χρησιμοποιείται σε offline mode).
- **Main Flow**:
  1. Ο αγρότης ανοίγει τη λειτουργία Scanner στην εφαρμογή.
  2. Στρέφει την κάμερα προς το φυτό που θέλει να ελέγξει.
  3. Πατάει το κουμπί για λήψη εικόνας.
  4. Ο αλγόριθμος αναλύει την εικόνα και παρέχει αποτελέσματα.
- **Alternative Flow**:
  - Εάν δεν υπάρχει σύνδεση στο διαδίκτυο, η εφαρμογή αποθηκεύει την εικόνα για ανάλυση όταν επιστρέψει το σήμα.
- **Post-conditions**:
  - Η διάγνωση αποθηκεύεται στα δεδομένα της εφαρμογής.
- **Edge Cases**:
  - Κακή ποιότητα εικόνας λόγω γαντιών ή χαμηλού φωτισμού: Η εφαρμογή ζητά νέα λήψη.
  - Λήψη εικόνας άλλου αντικειμένου: Εμφανίζεται μήνυμα "Μη αναγνωρίσιμο".

## Use Case 4: Offline Mode
- **Actor**: Αγρότης
- **Preconditions**:
  - Δεδομένα είχαν συγχρονιστεί πρόσφατα στο κινητό.
- **Main Flow**:
  1. Ο αγρότης ανοίγει την εφαρμογή χωρίς σύνδεση στο διαδίκτυο.
  2. Πρόσβαση σε αποθηκευμένα δεδομένα, όπως εγγραφές ημερολογίου ή διαγνώσεις ασθενειών.
- **Alternative Flow**:
  - Αν η τοπική βάση δεν έχει αποθηκευμένα δεδομένα, εμφανίζεται μήνυμα «Μη διαθέσιμα δεδομένα».
- **Post-conditions**:
  - Ενέργειες που εκτελέστηκαν offline συγχρονίζονται όταν επιστρέψει το σήμα.
- **Edge Cases**:
  - Η συσκευή έχει χαμηλή μπαταρία: Εμφανίζεται ειδοποίηση για φόρτιση πριν τη χρήση.
